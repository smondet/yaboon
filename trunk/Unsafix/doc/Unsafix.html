Nanosleep


type nanosleep_status = [ `error_EFAULT | `error_EINVAL | `interrupted of int * int | `ok ]
Return type of nanosleep and nanosleep_float
val nanosleep : int -> int -> nanosleep_status
High resolution sleep. Call nanosleep seconds nanoseconds, if the call is interrupted (e.g. by a signal), it will return `interrupted (secs,nanos) with the remaining time.
val nanosleep_float : float -> nanosleep_status
Same as nanosleep but with a float argument
